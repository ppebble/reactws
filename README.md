# Nexmore ERP 고도화 
- React 18+
- Node 20.11.+
- ES6 & TypeScript 5.3.+
- ESLint & airbnb
- React-Query & [데이터 처리방식](https://pozafly.github.io/react-query/mutation-after-data-update/)
- pnpm (Package Manager)
- Vite (Bundling)


## install pnpm
```bash
npm install -g pnpm
```

## install package
```bash
pnpm install
```

## start
```bash
pnpm run dev
```
